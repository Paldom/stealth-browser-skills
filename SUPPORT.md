# Support

Thanks for using Stealth Browser Skills! Fastest routes to help:

- **A skill misbehaves or never triggers** — open a
  [bug report](../../issues/new?template=bug_report.md). Include the skill name,
  the exact prompt you used, and what Claude did vs. what you expected.
- **Idea for a new skill** — open a
  [skill proposal](../../issues/new?template=skill_proposal.md) first; see
  [CONTRIBUTING.md](CONTRIBUTING.md) for what makes a good one.
- **Usage questions** — check the README [Quick start](README.md#quick-start) and
  [docs/](docs/) first; if still stuck, open an issue via
  [the chooser](../../issues/new/choose) (or a Discussion, if enabled).
- **Security problems** — **never** open a public issue; follow
  [SECURITY.md](SECURITY.md).

Only the latest release on the default branch is supported. This project is
maintained on a best-effort basis — clear, reproducible reports get answered first.
